## Consistency Checklist
- [ ] Magic rules are internally consistent
- [ ] Same power works the same way each time
- [ ] Limitations are clear and respected
- [ ] Magic isn't used to solve unsolvable problems
- [ ] Magic creates new plot complications, not just solutions
